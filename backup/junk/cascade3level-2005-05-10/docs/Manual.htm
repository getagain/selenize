DynamicDropDown - The class is to Fetch values from the DB and display in 3 ListBox.

Explaination: The Class can be used to Fetch the record for the database and load it in different list boxes, where are the Values of the child will be depended to the Parent. Here we have 3 Levels, Like illustrated below .

Type -----------------> Brand -------------> Model.

Its like when we select the Type respective Brand list will be loaded, when the use makes the choice in the Brand dropdown then the Model is made avaiable based on the previous box value.

Line 21 : Assign a NAME for the Form which we will be using in this Example Script
Line 22 : Call to the DataFetch method, which will Fetch the data from the Tables.
Line 23 : Call to the CreateJavaScript method,which will dynamically creates the Javascript which loads the Data to the List Box
Line 24 : Creates a New form on this Example Script Page

Line 26 : Close the Form

You can use the Mysql Schema to Create the Data Structure

#
# Table structure for table `type`
#

CREATE TABLE `type` (
`type_id` smallint(9) NOT NULL auto_increment,
`type_name` varchar(255) NOT NULL default '',
PRIMARY KEY (`type_id`)
) TYPE=MyISAM AUTO_INCREMENT=3 ;


#
# Table structure for table `brand`
#

CREATE TABLE `brand` (
`brand_id` smallint(9) NOT NULL auto_increment,
`brand_name` varchar(255) NOT NULL default '',
`type_id` smallint(9) NOT NULL default '0',
PRIMARY KEY (`brand_id`)
) TYPE=MyISAM AUTO_INCREMENT=4 ;


#
# Table structure for table `model`
#

CREATE TABLE `model` (
`model_id` smallint(9) NOT NULL auto_increment,
`model_name` varchar(255) NOT NULL default '',
`brand_id` smallint(9) NOT NULL default '0',
PRIMARY KEY (`model_id`)
) TYPE=MyISAM AUTO_INCREMENT=4 ;
